## What I learned

- A system prompt tells the LLM how it should behave.
- The system prompt is stored as the first message in the `messages` list.
- system prompts can conflick each other..
- The user never sees the system prompt.
- Every API request includes the system prompt along with the conversation history.
- A system prompt changes the assistant's behavior but does not make it a real expert.
- a chatbot is something which will only answer questions based on predefined rules where as an ai agent is autonomous, will think, use external tools and execute workflows to achieve a specific goal without human intervention.
- a tool is something that llm use to get answer for something that it doesn't know..
