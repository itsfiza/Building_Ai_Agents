# 🤖 Agentic AI Learning Journey

Welcome to my Agentic AI learning repository!

This repository documents my journey of learning Agentic AI from scratch by building projects step by step instead of simply copying tutorials.

The goal is to understand how AI agents work internally, including Large Language Models (LLMs), APIs, conversation memory, system prompts, tool calling, reasoning, planning, and multi-agent systems.
